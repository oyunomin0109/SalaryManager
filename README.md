## 勤務時間給料管理システム
バイト先に納入した勤務時間給料管理システム<br>
勤務時間を計算し、その時給換算の給料額を表示するAndroidアプリです。<br>
計算した結果(時間、給料額)はデータベースに保存することができ、保存したデータベースを閲覧することも、出力することも可能です<br>
